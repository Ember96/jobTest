Salute

This app implements an User Login and Register and also a Job Offers CRUD

The point is to handle situations where only certain power users can handle CRUD options like Update, Create and Delete
It implements Fluent Validation and uses Angular Material Design to make it more visually confortable
It also implements a JWT Token to handle the authentication
It uses a SQLite Server Database
You can find api docs thanks to swagger in: http://localhost:5266/swagger/index.html

I really hope it meets your requirements
